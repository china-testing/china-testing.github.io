删除包含某个值的行

现有df = pd.DataFrame([[1,0,2],[1,2,3],[0,1,2],[4,5,6]])，请删除其中包含0的行。

参考答案：

>>> import pandas as pd
>>> df = pd.DataFrame([[1,0,2],[1,2,3],[0,1,2],[4,5,6]])
>>> df
   0  1  2
0  1  0  2
1  1  2  3
2  0  1  2
3  4  5  6
>>> df[(df != 0).all(1)]
   0  1  2
1  1  2  3
3  4  5  6
