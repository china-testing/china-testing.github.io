    最早的Python版本就有string模块。 之前在这个模块中实现的许多函数已经移至str对象的方法。 string模块保留了几个有用的常量和类，用于处理str对象。

    代码地址

    函数

    capwords()的将字符串中所有单词的首字母大写。

    >>> import string
>>> t = "hello world!"
>>> string.capwords(t)
'Hello World!'
>>> t
'hello world!'
>>> t.capitalize()
'Hello world!'
>>> t
'hello world!'

    结果等同于先调用split()，把结果列表中各个单词的首字母大写，然后调用join()合并结果。

    因为str对象已经有capitalize()方法，该函数的实际意义并不大。
