    函数

    capwords()的将字符串中所有单词的首字母大写。

    >>> import string
>>> t = "hello world!"
>>> string.capwords(t)
'Hello World!'
>>> t
'hello world!'
>>> t.capitalize()
'Hello world!'
>>> t
'hello world!'

    结果等同于先调用split()，把结果列表中各个单词的首字母大写，然后调用join()合并结果。
